​ ls -lh按照文件大小单位来显示

​ ls -a显示所有文件包括隐藏文件（隐藏文件是以.开头的）

4.2 cd切换目录的命令

​ cd 目录名

​ 如果目录名称过长，可以输入前几个字母后使用tab键或esc键将其补齐。

​ cd .. 是返回上一级目录

​ cd - 返回上一次操作的目录

​ cd ~ 返回当前用户的家目录（cd后面什么都不加，也可以实现）

​ cd / 打开根目录

​ 在/home/linux目录下，想要进入/etc目录

​ 可以通过绝对路径的方式 ： cd /etc

​ 可以用相对路径方式：cd ../../etc

​ cd 绝对路径：从根目录开始访问

​ cd 相对路径：通过目录之间的位置关系访问

4.5、创建文件

​ touch命令

​ 如果文件不存在就创建空的普通文件

​ touch yapingnihao

​ 如果文件存在就更新文件的时间戳，不会改变文件的内容

​ 第一次touch -rw-rw-r– 1 linux linux 6 六月 15 15:38 yapingnihao

​ 第二次touch -rw-rw-r– 1 linux linux 6 六月 15 15:39 yapingnihao

​ 4.6、mkdir创建目录（make directory）

​ mkdir jiaweinihao 在当前目录创建一个jiaweinihao的目录

​ 创建的目录不可以和文件和文件夹重名

​ mkdir d1 d2 d3 可以创建多个目录，这些目录是在同一级的。

​ mkdir -p 1/2/3 创建具备层级关系的1 2 3这些目录

​ 4.7、删除目录（remove directory）

​ rmdir只能用来删除空目录

​ 4.8、rm命令（remove）

​ rm 1.c 删除当前目录下的1.c

​ rm *.c 删除所有以.c结尾的文件（ *是通配符，表示所有）

​ 删除目录

​ rm -r 1 删除1目录（-r递归删除）

​ rm test* -rf （-f强制删除）

​ rm * -rf 删除所有的当前目录下文件文件夹

​ 4.9、复制命令cp（拷贝）

​ 复制文件

​ 复制不改名cp 源文件的路径/名字 目标路径

​ cp ../../a.c 3/ 将上上目录中的a.c文件，复制到3这个目录中，

​ 复制并改名 cp 源文件的路径/名字 目标路径/新名字

​ cp test/a.c 3/4/b.c 将test目录中的a.c复制到3/4/目录后命名为b.c

​ 复制目录

​ cp 路径/原目录 目标路径 -r （递归拷贝）

​ cp 路径/原目录 目标路径/新名 -r 复制并重命名

​ 4.10、移动mv命令move

​ 移动

​ mv 路径/源文件 目标路径

​ mv 路径/目录 目标路径

​ mv 1/2/ydhou 2/ 把ydhou 移动到2目录中

​ mv 3/ydhou 5/ccc 把yidhou移动到5目录中并改名为ccc

​ 重命名

​ mv 旧的名字 新的名字

​ linux@ubuntu:~/jiaweinihao$ touch aaaa

​ linux@ubuntu:/jiaweinihao$ ls
​ 666666 aaaa
​ linux@ubuntu:/jiaweinihao$ mv aaaa zzzz

4.11、echo打印命令

​ echo helloxinzong 将helloxinzong打印在终端上

​ echo nihaojiawei > a.c 将nihaojiawei输出重定向到a.c文件中（覆盖之前a.c的内容）

​ echo xinzongnihao >> a.c将xinzongnihao输出重定向到a.c文件中（追加在文件内容的末尾，不覆盖）

​ echo yingying >> a.c
